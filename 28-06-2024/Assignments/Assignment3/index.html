HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages and web applications. It consists of a series of elements that define the structure of content on a webpage. Each HTML element is represented by a tag and may include attributes that provide additional information or control how the element behaves or appears.

CSS

CSS (Cascading Style Sheets) is the language used to describe the presentation of HTML and XML documents, including the layout, colors, fonts, and other visual aspects.
